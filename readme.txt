== Description ==

This plugin will allow you to add CSS style definitions to your entire site, or
individual posts or pages. All of your definitions are added last so they can
override any other styles, even styles from other plugins.

= Features =

* Add definitions sitewide.
* Add definitions on individual posts and pages.

== Installation ==

1. Upload `style-press` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Open the Style Press page under the Appearance Menu to configure.

== Frequently Asked Questions ==

= Where is the Style Press Settings Page =

You will find a page to manage the default settings and add sitewide styles
under the "Appearance" menu on your Dashboard.
